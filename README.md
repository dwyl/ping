# ping

The easy way to ping (wake) a Heroku App from it's idle slumber.

## Installation

Install by adding `ping` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:ping, "~> 0.1.0"}
  ]
end
```

Documentation available at [https://hexdocs.pm/ping](https://hexdocs.pm/ping).
